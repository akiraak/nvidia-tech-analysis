# NVIDIA Network Research Project

NVIDIA の技術について調査し、ドキュメント化するプロジェクト。

## プロジェクト概要

- NVIDIA のネットワーク関連技術を中心に調査・整理する
- 調査結果はマークダウン形式でドキュメント化する

## ディレクトリ構成

```
nvidia-network/
├── CLAUDE.md          # プロジェクト設定・ルール
├── TODO.md            # 未完了タスク
├── DONE.md            # 完了タスク
├── research/          # 調査ノート（マークダウン）
└── site/              # HTML 出力用（まとめ・公開向け）
```

- `TODO.md` — 未完了のタスクを記載する。完了したら `DONE.md` に移動する
- `DONE.md` — 完了済みタスクの記録
- `research/` — 調査の生データ・メモ。トピックごとにマークダウンファイルを作成する
- `site/` — research の内容を整理・まとめて HTML 化したもの

## ルール

- ドキュメントは日本語で記述する
- 技術用語・製品名は原文（英語）のまま使用する
- 調査ソースの URL を明記する
- ファイル名は英語のケバブケース（例: `gpu-architecture.md`）を使用する
- `research/` のファイルは調査の詳細・メモを自由に記述する
- `site/` の HTML は research の内容をまとめた清書版とする
