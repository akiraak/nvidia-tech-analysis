NVSwitch

NVLink ファブリックをノード規模に拡張。NVL72 で 72 GPU を 130 TB/s で non-blocking 接続。576 GPU で 1 PB/s の合計帯域幅。

NVLink Spine

ラック内 72 GPU を接続する物理インターコネクト構造体。130 TB/s 持続スループット。NVLink Fusion スイッチチップで 14.4 TB/s non-blocking。

NVLink Fusion

カスタム CPU / XPU を NVLink に統合可能にするオープンプラットフォーム。MediaTek, Marvell, Fujitsu, Qualcomm 等がパートナー。

NVLink-C2C

チップ間直接接続。Grace-Hopper: 900 GB/s、Vera Rubin: 1.8 TB/s。CPU-GPU 間のコヒーレント接続を実現。

CUDA

GPU プログラミングの基盤。C++, Python, Fortran 対応の並列計算プラットフォーム。

DOCA

BlueField DPU / SuperNIC 向け SDK。RDMA, ネットワーク, セキュリティ, ストレージの各 SDK を含む。

NCCL

GPU 間 collective communication ライブラリ。AllReduce, AllGather 等の分散 AI トレーニング通信を最適化。最新: v2.26。

Magnum IO

NCCL, NVSHMEM, GPUDirect, SHARP, UCX を統合したデータセンター IO スタック。

その他の主要技術

データ移動最適化、In-Network Computing、シミュレーション基盤

GPUDirect RDMA / Storage

GPU メモリへの直接データ交換。RDMA による GPU 間通信と、ストレージからの直接ロードで CPU オーバーヘッドを排除。

SHARP

InfiniBand スイッチ上の In-Network Computing。AllReduce 5 倍、Barrier 9 倍の高速化。ネットワーク転送量を半減。

Silicon Photonics / CPO

光通信をスイッチ ASIC に統合。3.5 倍の電力効率、63 倍の信号完全性。数百万 GPU の接続を可能に。

BlueField Astra

BlueField-4 DPU のセキュリティアーキテクチャ。East-West ファブリックの管理・ポリシー適用を拡張。

NVIDIA Air

クラウドホスト型ネットワークシミュレーション。データセンターネットワークのデジタルツインを構築・検証。

NVIDIA Omniverse

物理 AI シミュレーション基盤。産業デジタルツイン、ロボティクスシミュレーション向けプラットフォーム。
